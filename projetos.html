Meus projetos pessoais

Sistema bancário com python

Neste código é aplicado diversas formas de se criar um sistema bancário no qual adiciona um novo usuário colocando seu cpf como único para próximos cadastros,salva as alterações no saldo do usuário após as transações de depósito e saque.Além de todas estas funções também é especificado certas condições como limite para saque e quantas vezes é possível realizar o saque.

Excel sem stress com Pandas

Usando a biblioteca pandas este projeto demonstra como podemos analisar dados em planilhas Excel interagindo de diversas formas como:Filtrar dados,criar colunas novas,calcular estatísticas e etc.Esta é uma alternativa para analisar dados de planilhas extensas usando da lógica de programação como aliada para este processo.

Decodificador de textos

O decodificador tem como principal função proteger as mensagens digitadas usando da criptografia protegendo assim a privacidade das informações.Além da criptografia também foi criada uma função para descriptografar as mensagens podendo assim ter maior controle do que o usuário deseja comunicar posteriormente como mensagem.
